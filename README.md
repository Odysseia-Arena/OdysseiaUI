# OdysseiaUI

本项目是一个大模型前端。

## 开发模式

```bash
npm run dev
```

## 用户模式

```bash
npm run build
npm run preview
```

## 手机访问（局域网）

1. 在电脑上部署项目：
   ```bash
   npm run build
   npm run preview
   ```

2. 查看电脑的局域网 IP 地址（Windows: `ipconfig`，Mac/Linux: `ifconfig`）

3. 在手机浏览器中访问：`http://电脑IP地址:4173`（或预览服务显示的端口）

4. 确保手机和电脑连接在同一局域网（WiFi）下

> 提示：公网部署类似，将服务部署到公网服务器后，通过公网 IP 或域名访问即可。
